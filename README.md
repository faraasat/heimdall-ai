# Deriv Aegis MVP

Demo front end for the autonomous pentesting MVP. The UI focuses on safe, authorized scanning with human oversight and real-time agent activity.

## Getting started

Install dependencies and run the dev server:

```bash
yarn install
yarn dev
```

Open http://localhost:3000 in your browser.

## Project notes

- App Router with TypeScript and Tailwind.
- Demo mode only; results are simulated.
- LLM routing: Groq primary, Gemini/HF fallback.
